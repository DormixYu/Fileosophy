Quick Start
快速开始

首次使用Fileosophy，三步即可开始管理项目

设置个人档案 首次启动时，系统会弹出欢迎向导。选择一个头像（支持 emoji 或上传图片），输入你的名字，点击「开始使用」。后续可在「设置 → 用户资料」中修改。
配置状态与分类 进入「设置 → 项目配置」页面，在「状态管理」中添加项目状态（如：规划中、进行中、已暂停、已完成、已取消），每个状态可自定义颜色。在「分类管理」中创建分类体系，配置编码前缀和识别关键词。
创建第一个项目 回到项目列表页面，点击右上角的「新建项目」按钮（或按 Ctrl+Shift+N），填写项目名称、选择状态和分类，即可创建项目。项目卡片会出现在对应状态的看板列中。

Kanban Board
项目看板

核心工作区，以看板和表格双视图管理所有项目

看板视图

  • 拖拽变更状态 — 按住项目卡片拖拽到其他状态列，即可变更项目状态。拖拽时会自动记录状态变更历史。
  • 卡片信息 — 每张卡片显示：项目编号、名称、分类标签、标签列表。
  • 双击打开文件夹 — 双击卡片可直接打开项目关联的文件夹。
  • 底部操作 — 卡片底部有快捷按钮：打开文件夹、设置共享、编辑项目。
  • 批量选择 — 点击卡片左上角的复选框选中项目，支持批量状态变更。

表格视图

右侧表格以列表形式展示项目，支持双击打开文件夹、点击操作按钮编辑或共享。表格列可在「设置 → 项目配置」中配置。

工具栏

  • 搜索 — 输入关键词实时过滤，匹配项目名称、编号。
  • 状态快捷筛选 — 点击状态按钮快速切换。
  • 高级筛选 — 可按分类、日期范围组合筛选。

项目操作

操作方式

删除项目选中后点击删除按钮，支持批量删除

Gantt Chart
甘特图

时间线视图，直观把控项目起止时间和进度

视图说明

甘特图以 HTML 渲染，每个项目一行任务条。任务条使用项目的状态变更历史渲染多段颜色，展示项目在不同阶段的时间分配。今天日期用红色竖线标识。

工具栏操作

  • 缩放 — 按住 Ctrl 滚动鼠标滚轮缩放，支持日、月、年三种视图粒度。
  • 筛选 — 可按分类或状态过滤项目。
  • 回到今天 — 点击「今天」按钮一键定位。

交互操作

  • 悬停 — 鼠标移到任务条上显示项目详情 Tooltip。
  • 里程碑 — 可在甘特图上添加和管理项目里程碑。
  • 状态历史 — 可查看和编辑项目状态变更历史。

局域网共享

在局域网内共享项目文件夹，支持密码认证和文件浏览

主机端操作

  1. 进入「共享」页面，点击「开启共享」选择文件夹并设置密码
  2. 查看已连接的客户端列表和活动日志
  3. 客户端可浏览共享目录和下载文件

客户端操作

  1. 进入「共享 → 连接他人」页面
  2. 输入主机地址和密码连接
  3. 浏览远程文件目录，下载所需文件
  4. 也可上传文件到主机端

mDNS 自动发现

局域网内的 Fileosophy 实例会通过 mDNS 协议自动互相发现，无需手动输入 IP。发现的对等节点显示在「局域网发现」面板中。

注意 — 所有设备需在同一局域网内。传输数据通过 TCP 协议发送，建议在可信网络环境中使用。

FAQ
常见问题

使用过程中可能遇到的问题与解答

如何备份项目数据？ ▼
所有项目数据存储在本地 SQLite 数据库文件（fileosophy.db）中，位于应用数据目录下。进入「设置 → 数据管理」可使用导出功能备份所有项目数据（JSON 格式）。也可直接复制数据库文件完成备份。
局域网共享时其他设备无法连接？ ▼
请确保：① 所有设备在同一局域网内；② 防火墙未阻止应用的网络访问；③ 主机端已正确开启共享并设置了密码。如果 mDNS 自动发现不生效，可以手动输入主机的 IP 地址和端口进行连接。
如何自定义看板的列（状态）？ ▼
进入「设置 → 项目配置 → 状态管理」，可以添加、编辑、删除和排序状态。每个状态支持自定义颜色和名称。修改后看板视图会自动更新。注意：删除状态前请确保该状态下没有项目。
如何切换深色模式？ ▼
进入「设置 → 关于」，在主题选项中选择「深色模式」即可。也可以选择「跟随系统」让应用自动匹配操作系统的主题设置。
支持哪些操作系统？ ▼
Fileosophy·飞序基于 Tauri 2.x 构建，支持 Windows、macOS 和 Linux 三大平台。推荐使用 Windows 10+ 或 macOS 12+ 以获得最佳体验。
